# RPEssentials
An essential for any Casual/Serious-RP server: especially starwarsrp
---
* Has /comms system
* Has /looc 
* Provides /hcomms with /comms | 'Hacked Comms'
* Has an animation system with passive action
	* Easily Changeable, Including
		* /me salutes
		* /me waves
		* /me dances
		* /me cheers
		* /me laughs
		* /me bows
